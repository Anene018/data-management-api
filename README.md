## Project setup

```bash
$ npm install

```

## Docker setup

```bash
$ docker compose up

```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

```
